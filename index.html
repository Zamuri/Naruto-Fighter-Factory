Project Config

Bisa pakai file DEF/CMD/CNS/AIR/SFF/SND atau ZIP project.
Project routing: sprite, audio, text, dan inspector akan terpisah otomatis ke mode terkait. Export di luar tab menyimpan semuanya menjadi satu project.
Tab ini dipakai untuk mengontrol isi file DEF dan draft project lain agar lebih seperti workspace Fighter Factory.

SWF Extractor

File SWF bisa berbeda dari file project.
SWF bitmap dan audio akan diekstrak dari tag asli. Vector shape/timeline SWF belum bisa dirender sempurna tanpa engine Flash/Ruffle. CWS zlib didukung lewat browser modern, ZWS/LZMA belum.

MUGEN Builder

Builder membuat project MUGEN siap compile: PNG + .sffdef untuk SprMake2, .air, .def, .cmd, dan .cns draft. Ini lebih rapi untuk workflow converter.

Current Rules

- Sprite file akan disusun dari input/output PNG aktif.
- Group dan image mengikuti tabel Sprite Edit.
- DEF mengikuti Project Config.
- AIR draft dibuat otomatis per group action.
